# My personal webpage

A personal portfolio website built with React, TailwindCSS and Vite. You can check it out here: [alkor-istar.github.io](https://alkor-istar.github.io)

## Features

* Responsive design
* Modular design with React components for each section
* Smooth scrolling with transitions

## Technologies used

* React
* TailwindCSS
* Vite

## Deployment

* The website is deployed to GitHub Pages
* The website is built and deployed automatically using GitHub Actions
